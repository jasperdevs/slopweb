# Slopweb App

This folder contains the local HTTP server, static browser shell, and page-generation runtime used by the `slopweb` CLI.

Run it through the package entrypoint from the repository root:

```powershell
pnpm start
```

Run the app server directly when you are already in this folder:

```powershell
node server.js
```

See the root [README](../README.md) for install, local model setup, Codex OAuth, and CLI usage.
